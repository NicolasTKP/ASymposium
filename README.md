# Data and Codes
## Directory Structure
```kotlin
ASymposium/
│── Data/
│   ├── Archive/ // Containing special data files
│   ├── Others/ // Containing uncategorized data files
|   ├──Py_output/ // Containing all output of the Python file
|   ├──R_output/ // Containing all output of the R script file
|   ├──Raw_Data/ // Containing the raw data without cleaning
│── Model/ // Containing all trained models, training history, and scaler files
│── Rscript/ // Containing all R script files
│── Utils/ // Containing all Python files used for utilities
│── WebScrap/ // Containing all Python files used for web scraping
│── expt_loss_calculator.py // The Python file used for expected loss calculation
│── index.html // The HTML file for the station mapping and flood range overview (as well as the flood probability)
```
